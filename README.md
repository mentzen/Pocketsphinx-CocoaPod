## Requirements

Sphinxbase-CocoaPod gets added Automatically

## Installation

Pocketsphinx-CocoaPod is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "Pocketsphinx-CocoaPod"
```
